Tools build with CL.js

Builder Widget

The Builder Widget is a tool to help you embed content from feeds.library.caltech.edu and your own website. It generates example CSS, HTML and JavaScript based on a set of feed sources you choose.

Search Widget

The Search Widget is a tool to help you embed a search in your website for content coming from feeds.library.caltech.edu. Like the builder widget it generates example CSS, HTML and JavaScript needed to include search in your own web page.
